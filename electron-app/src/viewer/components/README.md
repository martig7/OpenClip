# Viewer Components

This folder contains reusable React components for the video viewer interface.

## Key Files

- **VideoPlayer.jsx** - Core video playback component
- **ZoomTimeline.jsx** - Timeline with zoom functionality
- **StorageTreemap.jsx** - Visual representation of storage usage
- **Sidebar.jsx** - Navigation sidebar for viewer
- **ClipControls.jsx** - Controls for clip manipulation
- **AudioWaveformTrack.jsx** - Audio waveform visualization
- **Timeline.jsx** - Base timeline component
- **Modal.jsx** - Reusable modal dialog component
- **ReencodeModal.jsx** - Modal for video re-encoding options

## Description

These components provide the UI building blocks for the video viewer. They handle video playback, timeline interaction, storage visualization, and user interactions.

For detailed information about each component, refer to the individual file implementations.
