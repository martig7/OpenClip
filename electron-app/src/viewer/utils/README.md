# Viewer Utilities

This folder contains utility functions for the video viewer module.

## Key Files

- **treemapUtils.js** - Utilities for treemap calculations and rendering
- **storageColors.js** - Color schemes for storage visualization
- **utils.js** - General utility functions for viewer functionality

## Description

These utilities support the viewer components with data processing, color mapping, and helper functions needed throughout the viewer module.

For detailed information about each utility, refer to the individual file implementations.
