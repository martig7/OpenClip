# Main Pages

This folder contains the primary page components of the application.

## Key Files

- **GamesPage.jsx** - Main page for game management and recording configuration
- **SettingsPage.jsx** - Application settings page
- **EncodingPage.jsx** - Video encoding configuration page
- **AddGameModal.jsx** - Modal for adding new games (shared)
- **EditGameModal.jsx** - Modal for editing games (shared)
- **SceneAudioSourcesCard.jsx** - Card for audio source configuration (shared)
- **WatcherStatusCard.jsx** - Card for watcher status display (shared)

## Subfolders

- **[games](./games/README.md)** - Game-specific components and utilities

## Description

These pages form the main navigation structure of the application. GamesPage handles game configuration and recording setup, SettingsPage manages application preferences, and EncodingPage configures video encoding options.

For detailed information about game-related components, see [games/README.md](./games/README.md).
