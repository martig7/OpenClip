# Video Viewer

This folder contains the video viewer module - a comprehensive UI for browsing, playing, and managing recorded videos and clips.

## Key Subfolders

- **[pages](./pages/README.md)** - Main viewer page components (StoragePage, ClipsPage, RecordingsPage)
- **[components](./components/README.md)** - Reusable UI components for video playback and visualization
- **[utils](./utils/README.md)** - Utility functions supporting viewer functionality

## Additional Files

- **viewer.css** - Styles for the viewer module
- **apiBase.js** - API base configuration for viewer requests
- **utils.js** - General viewer utilities

## Description

The viewer module provides the frontend interface for browsing and playing recorded content. It integrates with the backend recording services to display video files, manage storage, and provide playback controls.

For detailed information about pages, components, or utilities, see the respective README files in the subfolders.
