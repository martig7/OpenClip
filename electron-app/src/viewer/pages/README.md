# Viewer Pages

This folder contains the main page components for the video viewer section of the application.

## Key Files

- **StoragePage.jsx** - Page for managing storage and viewing storage statistics
- **ClipsPage.jsx** - Page for viewing and managing video clips
- **RecordingsPage.jsx** - Page for viewing and managing recordings

## Description

These pages form the navigation structure of the video viewer interface. Each page handles specific video-related functionality including storage management, clip organization, and recording playback.

For detailed information about each page component, refer to the individual file implementations.
