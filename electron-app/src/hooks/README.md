# React Hooks

This folder contains custom React hooks for state management and application logic.

## Key Files

- **useAddGameModalState.js** - State management for the add game modal
- **useAudioSourcesState.js** - State management for audio sources
- **useTrackState.js** - State management for playback track information
- **useGameWatcherState.js** - State management for game watcher functionality
- **useToastState.js** - State management for toast notifications

## Description

These hooks encapsulate reusable state logic and side effects used throughout the application. They provide centralized state management for modals, audio sources, playback tracks, game watching, and notifications.

For detailed information about each hook, refer to the individual file implementations.
