# OpenClip OBS Plugin — Include

This folder contains OBS Studio SDK headers used for building the plugin.

## Files

| File | Description |
|------|-------------|
| `obs.h` | Core OBS API (video, audio, sources) |
| `obs-module.h` | OBS plugin loading and module API |
| `obs-frontend-api.h` | OBS frontend API (scenes, recording, profiles) |

## Source

These headers are copied from the OBS Studio SDK (version 30.0+). They are included in the repository for convenience but should be kept in sync with the OBS version the plugin targets.
