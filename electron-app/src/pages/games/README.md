# Game Pages

This folder contains game-specific page components and utilities for the games management section.

## Key Files

- **AddGameModal.jsx** - Modal for adding new games to track
- **EditGameModal.jsx** - Modal for editing existing game configurations
- **SceneAudioSourcesCard.jsx** - Card component for managing audio sources per scene
- **WatcherStatusCard.jsx** - Card displaying the current watcher/status information
- **audioSourceUtils.jsx** - Utilities for handling audio source configurations
- **WindowPicker.jsx** - Component for selecting application windows to track
- **AudioSourceDropdown.jsx** - Dropdown for selecting audio sources

## Description

These components provide the UI for managing game recording configurations. They handle adding and editing games, selecting windows to monitor, and configuring audio sources.

For detailed information about each component, refer to the individual file implementations.
